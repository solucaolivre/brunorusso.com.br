Hyperterminal no Linux

Bruno T. Russo Feb 24, 2009 2009-02-24T02:00:51+00:00
1 min

Toda vez que precisava configurar um Switch via porta SERIAL, lá ia eu usar uma máquina Windows, para usar o hyperterminail. Usava esse, pois nunca fui com a cara do minicom. Simplesmente não gosto de usá-lo, pois todas as vezes que precisei utilizá-lo apanhei e muito.

Mas cansei de utilizar uma máquina Windows para isso, então...procura aqui..acessa ali...até que!

Eureka!

Encontrei um excelente hyperterminal para linux, chamado: C-kermit, ou apenas kermit.

A utilização do kermit é muito simples, segue um exemplo feito pelo: Marcelo B. Miola em seu blog.

Utilizando a interface serial do seu micro para se conectar a console de um roteador ou switch (salvar comandos abaixo no arquivo serial.txt).

log session serial.log
set line /dev/ttyS0
set speed 9600
set serial 8N1
set carrier-watch off
connect
close connection
quit

Após isso execute: # kermit serial.txt

Pronto! Você estará conectado em seu switch. Vou tentar fazer o pacote desse hyperterminal para o Slackware, mas não pometo...
