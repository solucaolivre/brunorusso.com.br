Ao tentar instalar o OCS Inventory em uma máquina CentOS com uma instalação mínina, a seguinte mensagem de erro é apresentada:

 +----------------------------------------------------------+
 | Checking for PERL Interpreter... |
 +----------------------------------------------------------+
 PERL Interpreter not found !
 OCS Inventory NG is not able to work without PERL Interpreter.
 Setup manually PERL first.
 Installation aborted !

Porém, o pacote PERL está instalado! Esse erro pode ocorrer devido a falta de um pocote chamado: which.

Para resolver esse problema, basta instalar o pacote.

#yum install which
