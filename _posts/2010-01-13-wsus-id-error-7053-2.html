A mensagem de erro com o ID 7053, ocorreu o WSUS 3.1.

Esse problema ocorreu devido a uma alteração na versão do Framework (uma espécie de "downgrade"). Isso pode ter ocorrido por diversos motivos.

Para resolver o problema, executei os seguintes passos.

  1. Iniciar >> Executar >> inetmgr
  2. Botão Direito no Default Web Site e selecione propriedades
  3. Na aba ASP.NET altere a versão de 1.4322 para 2.0.50727 em seguida click em OK
  4. Abra um prompt de comando e navegue até a pasta %windir%Microsoft.netframeworkv2.0.50727
  5. Execute aspnet_regiis –I
  6. Em seguida execute iisreset /restart
