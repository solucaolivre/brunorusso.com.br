Atualmente, a "modinha" é virtualização. Tudo que se ouve relacionado à tecnologia tem virtualização!!!

Muitos softwares estão disponíveis para que seja possível realizar a virtualização de Servidores. Porém, somente ter o software não é possível para virtualizar um servidor. É necessário um hardware capaz de suportar a virtualização.

A idéia de virtualizar é muito interessante, porém para que seja possível tomar a decisão acertada é necessário planejar. Algumas perguntas devem ser muito bem respondidas para que a virtualização ocorra de maneira adequada e planejada:

  • O que se deseja virtualizar?;
  • Qual o período de virtualização?;
  • Qual o hardware que será utilizado para a virtualização?;
  • Qual o software que será utilizado para a virtualização?;
  • Como será o esquema de contingência para essa máquina virtual, caso alguma pane ocorra?

Essas, são apeas algumas perguntas que devem ser muito bem respondidas.

Os softwares utilizados para a virtualização, possuem características bem parecidas, porém todos possuem vantagens e desvantagens em sua utilização.

 

 

Atualmente, para experiências pessoais utilizo o VirtualBox, que atualmente é da Sun Microsystem. Esse software é 100% livre e pode ser utilizado em diversos Sistemas Operacionais.

Utilizo o VirtualBox para virtualizar: máquinas Windows, Linux (slackware) e FreeBSD. Todas essas máquinas são virtualizadas com o VirtualBox em meu Sistema Operacional Slackware Linux .
