Lançado, oficialmente o FreeBSD 7.1.

Algumas novidades são:

  • O Ule scheduler é agora o padrão no kernel GENERIC para arquiteturas i386 e amd64. O agendador Ule multicondutores melhora significativamente o desempenho nos sistemas de muitas cargas.
  • Suporte para usar DTrace dentro do kernel foi importado de OpenSolaris. DTrace é traçado um quadro global dinâmico.
  • Um novo e muito melhorado NFS-Lock Manager (NLM) cliente.
  • Alterações no Boot loader permitem, entre outras coisas, iniciar o sistema a partir de dispositivos USB.
  • KDE atualizado para 3.5.10, GNOME atualizado para 2.22.3.

Para ver o ChangeLog dessa versão acesse o site oficial.
