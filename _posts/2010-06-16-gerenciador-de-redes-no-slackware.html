Um aplicativo para o gerenciamento de rede em modo gráfico é muito útil. O Slackware até então não tinha, ou melhor, possui um ótimo gerenciador para redes wifi, chamado wicd, que está disponível na seção extra do CD oficial.

Agora, graças ao esforço do Robby Workman, ele disponibilizou um SBo com o gerenciador de redes NetworkManager do Projeto Gnome.

NetworkManager is a set of co-operative tools that make networking simple and straightforward. Whether WiFi, wired, 3G, or Bluetooth, NetworkManager allows you to quickly move from one network to another: once a network has been configured and joined once, it can be detected and re-joined automatically the next time its available.

Estou utilizando esse gerenciador há cerca de 3 dias, e seu fincionamento está estável, porém algumas características precisam ser melhoradas, uma elas é a forma de inicialização, pois o gerenciador não está sendo carregado durante o login no ambiente gráfico!

Compilei e disponibilizei os pacotes para X64 em meu site:

http://www.brunorusso.com.br/pacotes/NetworkManager/

Precisamos ajudar no desenvolvimento dessa aplicação, para quem sabe o Pat, inclua ela em uma nova versão do Slackware.

Com o NetworkManager é possível, realizar conexões Cabo, Wifi, VPN, PPTP, ADSL.
