No período de 30/08/2007 à 01/09/2007, ocorrerá em Joinville - SC, ocorrerá o terceiro encontro da Comunidade Python Brasileira.

O evento, como o próprio site oficial do evento divulga, tem o

... objetivo de discutir tendências, o futuro da comunidade e da tecnologia, o evento PyConBrasil realiza a sua 3ª edição, contando com apoio da comunidade de software livre do país.
Essa é a oportunidade de conhecer poderosas ferramentas para desenvolvimento Web tais como Zope, Plone, Turbogears, Django, entre outros. Essas ferramentas permitem que qualquer programador desenvolva sites compatíveis com a Web 2.0 utilizando facilmente técnicas como Ajax.

Ok!! Mas Porque esse post, sendo que eu não sou programador Python???

Simples, python é uma dinâmica linguagem de programação orientaad a objeto! Funciona em qualquer Sistema Operacional (Windows, Linux/Unix, Mac OS X, OS/2, Amiga, Palm Handhelds e celulares Nokia). E tenho muito interesse em aprender a utilizá-la para criar rotinas que facilitam a vida!

Já cheguei a utilizar um script em Python pronto para ajudar a combater SPAM, e fiquei impressionado com a velocidade do programa final.

A grande vontade é re-aprender C/C++, atualmente o PHP é muito importante e facilita muito, porém uma terceira linguagem seria interessante aprender.

Por motivos óbvios não poderei participar do evento, mas gostaria e muito de saber o potencial que o python possui!!
