Recentimente começei a procurar um software para gerenciamento financeiro pessoal que fosse 100% Software Li vre. Há alguns anos havia utilizado o gnucash um bom software, desenv olvido por usuários do gnome e que roda em qualquer distribuição GNU/Linux, porém não me adaptei ao programa. Então começei a procurar um que pudesse atender às minhas necessidades.

Foi ai que lembrei de um site que possui uma lista de equivalência entre softwares para Windows e Software Livre. Essa lista é muito interessante, porém ainda está incompleta :(

A equivalência que a lista possui para softwares de gerenciamento financeiro pessoal é a da tabela a seguir:

Software Proprietário Software Livre
Quicken KMyMoney, GNUcash, Gnofin e Grisbi

Porém, o software que me chamou mais a minha atenção foi o KMyMoney .

O KMyMoney, foi desenvolvido com base no KDE, que ultimamente virou meu ambiente gráfico padrão (muitos dizem que o KDE é lento, porém isso é questão de customizaçãonão é o sistema que é lento, talvez é a falta de conhecimento em configurar o ambiente).

Para facilitar ainda mais a instalação, a compilação demora, criei um pacote para o slackware 11 que está disponível.

Com uma interface gráfica muito amigável, principalmente para quem já utilizou um software de gerenciamento financeieo pessoal muito conhecido. A usabilidade é incrível!

Com o KMyMoney, você pode gerenciar:

  1. Contas
    1. Correntes
    2. Poupança
  2. Carteira
  3. Aplicações
  4. Bens (casa, carro..)

Esses são apenas alguns exemplos do que o KMyMoney é capaz de fazer. Uma outra grande vantagem é que o software está 100% traduzido para o Português do Brasil. ;)

Um ponto fraco que achei no sistema foi que os diversos tipos de relatórios não possuem gráficos, porém existem uma grande quantidade de relatórios.
