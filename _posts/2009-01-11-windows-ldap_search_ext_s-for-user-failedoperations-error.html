windows [ldap_search_ext_s() for user failed][Operations error]

Tive que solucionar o erro acima, ao fazerm com que o Apache, faça a autenticação dos usuários em um Domínio controlado pelo Active Directory!

A maneira que consegui resolver o problema foi fazer a busca no AD, através de uma OU específica, exemplo

DC=BRUNORUSSO,DC=ETI,DC=BR

Ocorre o erro!

OU=USUARIO,DC=BRUNORUSSO,DC=ETI,DC=BR

Funciona!
