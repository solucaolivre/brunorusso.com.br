Já faz algum tempo que o google lançou o Software PICASA, um software para manipulação de imagens. No início não havia uma versão compatível com GNU/LINUX, porém após um tempo o mesmo software foi lançado para GNU/LINUX.

Já havia ouvido de várias pessoas que o software é muito bom e fácil de usar, porém nunca tinha tempo disponível para utilizá-lo e verificar a sua qualidade. ãté que resolvi instalar e testar.

A instalação é bem simples, pois você apenas baixa um arquivo .sh altera sua permissão para execução e pronto! Em menos de 2 minutos você terá o software instalado. Durante a instalação é criado um atalho na área de trabalho do kde e um outro atalho no menu programas/gráficos. Assim que termina a instalação o software já pode ser iniciado automaticamente.

Logo na primeira inicialização o software varre todo o disco (é claro que isso é configurável) a procura de arquivos de imagens. Dependendo da quantidade de imagens que você tem, essa tarefa irá demorar um pouco e um pouco considerável! Porém, assim que as imagens são localizadas, você já pode ir trabalhando com o software.

Para diminuir o tempo de varredura é aconselhável definir o diretório correto do local onde as imagens estão armazendas.

Uma vez que as imagens foram localizadas é hora de utilizar o software, e uma coisa que fiquei impressionado foi com a organização e com o lay-out do software. No lado esquerdo da tela ficam listadas as pastas e do lado direito/centro ficam as imagens em miniatura.

Ao selecionar qualquer pasta do lado direito, as imagens são exibidas instantâneamentes ao lado. Extremamente rápido!

Algumas ferramentas são extremamentes fáceis de utilizar, uma que posso destacar é a correção de olhos vermelhos. Com extrema facilidade é possível retirar o "vermelho do olho", isso é uma característica muito importante, pois usuários finais não possuem programas especialistas como GIMP e PHOTOSHOP, além do mais esses softwares não são tão fáceis de serem utilizados é preciso pelo menos uma noçao básica de imagem.

Uma outra característica muito importante é a facilidade de criar álbuns para serem publicados na web. Sem contar que é possível publicar as fotos diretamente no blogger do Google. Além disso também é possível girar as imagens, alterar o brilho, contraste tudo isso é muito mais de forma rápida e fácil.
