Testando o xfce

Bruno T. Russo Mar 13, 2007 2007-03-13T17:52:18+00:00
1 min

Hoje, começei a testar o xfce!

A versão que estou testando é a 4.2 que é a versão padrão do Slackware 11.

Um fato interessante que achei é a compatibilidade com os programas do kde e com uma interface, que se chama agua, que existe na qual é semelhante a interface do MAC.

A única coisa chata, pelo menos dessa versão é a forma de incluir os programas no menu. Um outro fato negativo que considero é que não existe um botão (do tipo "iniciar"), para usuários acostumados com interface Microsoft, isso torna o ambiente um pouco incômodo para o uso.

A velocidade do gerenciador é bem interessante, ele é rápido!

Amanhã irei instalar a última versão 4.4, segundo informações que obtive essa última versão possui muitas melhorias e algumas delas extremamente agradáveis, é isso que veremos.
