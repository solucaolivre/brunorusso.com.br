Select interessante para retornor do banco do CACIC, a relação dos conmputadores com número IP e os softwares utilizados.

 select a.te_nome_computador, a.id_ip_rede, c.nm_software_inventariado from computadores a, softwares_inventariados_estacoes b, softwares_inventariados c
where a.te_node_address = b.te_node_address and b.id_software_inventariado = c.id_software_inventariado

A saída é semelhante a:
