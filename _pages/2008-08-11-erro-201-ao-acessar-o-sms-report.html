Ao tentar acessar um relatório do SMS com um usuário diferente do Administrador do Site, a seguinte mensagem é exibida:

sms Report 201 is unavailable

Esse problema ocore devido a falta de permissão que o usuário possui para visualizar os relatórios. Isso pode ser facilmente corrigido incluindo a devida permissão para o usuário, na opção Security Rights. Para maiores detalhes, acesse: 887015.
