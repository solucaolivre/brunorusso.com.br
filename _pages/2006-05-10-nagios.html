Aqui estão alguns dos problemas que tive com o Nagios e as soluções possíveis!

1-) Ao tentar utilizar o plugin nrpe, a máquina remota retorna o seguinte erro: "Could not read request from client, bailing out...".

R: A máquina servidora está fazendo a requisição para a máquina remota utilizando autenticação via SSL. porém a máquina remota está configurada para não trabalhar com encriptação SSL. Para resolver esse problema, basta alterar o seguinte parâmetro do arquivo checkcommands.cfg:

define command{
command_name check_nrpe
command_line /usr/local/nagios/libexec/check_nrpe -H $HOSTADDRESS$ -c $ARG1$
}


para

define command{
command_name check_nrpe
command_line /usr/local/nagios/libexec/check_nrpe -H $HOSTADDRESS$ -c $ARG1$ -n
}
