Hoje em dia é muito comum em uma rede a existência de  de um ambiente heterogêneo de Sistemas Operacionais!

Muitas pessoas utilizam o Windows como Sistema Operacional para desktop e essas pessoas não possuem nenhum problema para acessar o Windows no Servidor, pois utilizam a opção do Acesso Remoto. E quando essas mesmas pessoas precisam acessar uma máquina Linux/Unix umas das inúmeras alternativas é utilizar o Putty (http://the.earth.li/~sgtatham/putty/latest/x86/putty.exe).

Mas para quem utiliza o Linux, assim como eu, uma opção interessante é utilizar o ssh para Servidores Linux e o rdesktop para acessar Servidores Windows.

O rdesktop é um programa de código fonte livre capaz de acessar um Terminal Remoto Windows, através do protocolo RDP.

Para seu funcionamento rdesktop necessita de um ambiente gráfico (Servidor X).

A instalação do rdesktop é super simples. A versão utilizada para essa instalação foi a 1.4.1. Vamos para a instalação:

  • cd /tmp
  • wget http://prdownloads.sourceforge.net/rdesktop/rdesktop-1.4.1.tar.gz?download
  • tar xzvf rdesktop-1.4.1.tar.gz
  • cd rdesktop-1.4.1
  • ./configure
  • make
  • make install

Pronto! O software já está instalado. Para acessar um servidor use:

  • rdesktop IP

rdesktop pode ser usado com inúmeras opções, para maiores informações sobre a ferramente consulte:

  • rdesktop –help
  • man rdesktop
