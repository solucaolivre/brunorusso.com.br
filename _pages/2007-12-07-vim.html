O Vim é um avançado editor de textos, que visa proporcionar o poder do Unix Editor, o Vi, com um conjunto mais completo de recursos. É útil se você já estiver usando o vi ou utilizando um editor diferente.

O Vim é altamente configurável, foi construído para permitir uma edição eficiente de um texto. Trata-se de uma versão melhorada do Editor vi distribuído com a maioria dos sistemas UNIX.

O Vim, também é conhecido como editor dos programadores, ele é muito útil para a programação. Porém, não é só para programadores. Vim é perfeito para todos os tipos de edição de textos.

Infelizmente, nem tudo são rosas...

O Vim não é um editor concebido para usuários leigos, pois trata-se de uma ferramenta, cuja utilização deve ser aprendida.

A sua licença é GPL, por isso pode ser distribuído livremente.

O mais interessante é que o VIM, pode ser executado em diversas plataformas. Para efetuar o download acesse o site original e obtenha a última versão do vim.

Nesse espaço vou colocar algumas dicas referentes a utilizaçãodo VIM.

1. Corrigir ortograficamente um texto através do VIM.

Dentro do VIM, em modo de edição, execute:

:!aspell -c -l pt_br %

Para instalar o dicionário em português no sistema, veja a dica completa no site da dicas-l.

2. Executar um comando externo, dentro do VIM.

Dentro do VIM, em modo de edição, execute:

:!COMANDO
